# KT Clip 用户手册

## 插件定位

KT Clip 是一款轻量、本地优先的文本剪贴板历史工具，目标是帮助用户快速找回和复用近期复制过的内容。插件专注于搜索、复制、收藏和清理等高频操作，无需注册账号，也不依赖云端服务。

它的主要特点包括：

- 复制文本后自动记录，减少重复查找和输入。
- 通过关键词实时搜索，快速定位需要的内容。
- 重要记录可以收藏，避免在批量清理时被删除。
- 数据仅保存在本机，由用户自行查看和删除。

## 打开插件

在 uTools 搜索框中输入以下任一指令：

- 剪贴板历史
- 剪贴记录
- Clipboard History

选择 KT Clip 后即可进入历史记录页面。

## 使用方法

### 查看和复制

复制文本后，记录会自动出现在列表顶部。单击记录内容或“复制”按钮，即可将该内容重新写入剪贴板。

### 搜索记录

在顶部搜索框输入关键词，列表会实时显示包含该关键词的记录。点击搜索框右侧的“×”可清空关键词。

### 收藏记录

点击记录右侧的星标按钮可以收藏或取消收藏。收藏记录会排在列表前面，且不会被“清空未收藏”操作删除。

### 删除记录

- 点击记录右侧的“×”可删除单条记录。
- 点击“清空未收藏”可删除全部未收藏记录。

删除后无法恢复，请确认记录已不再需要。

## 数据保存规则

- 插件仅记录文本，不记录图片和文件。
- 最多保留 200 条未收藏记录；超出后自动删除较早的未收藏记录。
- 收藏记录不会被自动清理。
- 相同文本再次复制时会更新为最近使用时间，不会创建重复记录。

## 隐私与安全

所有记录仅保存在本机 uTools 数据库中，不会上传至服务器。由于系统剪贴板可能包含密码、验证码、访问令牌、密钥等敏感内容，请在复制此类内容后及时删除对应记录。

## 常见问题

### 为什么找不到图片或文件记录？

当前版本仅支持文本剪贴板历史。

### 为什么记录数量不再增加？

插件最多保留 200 条未收藏记录。达到上限后，会自动删除较早的未收藏记录。

### 清空记录会删除收藏内容吗？

不会。“清空未收藏”只删除未收藏的记录。

### 数据会同步到其他设备吗？

不会。当前版本的数据仅保存在本机。
